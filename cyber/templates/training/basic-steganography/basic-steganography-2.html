Did you figure it out? Stego was hiding a secret message right under our nose, but we couldn't see it because the text was white.

Nice work, but rarely will it be that easy. The most common forms of steganography are done by tools that can encode and decode images for us. There are lots of different algorithms that can be used. Here are some exampes:

  • XOR the binary bits
  • Include extra data chunks that contain the hidden data but are not displayed
  • Every 10th pixel, replace it with data from the hidden data instead. This should change the image quality but may not be obvious if the image is only subtly changed

Tools

There are many tools that implement these algorithms and you can find them online, we'll be using: https://stylesuxx.github.io/steganography

Try decoding this blue stego image. You'll have to right click the image, save it to your computer, and then upload it to the website and press decode.
