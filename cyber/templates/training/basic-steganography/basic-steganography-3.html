File Extensions

Another way we might try to hide information inside a file, is to change the file extension.

File extensions let us know what type of file it is, for example:

  • .png is a PNG image file
  • .txt is a text file
  • .docx is a Microsoft Word document file
  • .mp3 is a an audio file

By changing the file extension, we can trick people into believing the file is one type of file (for example, an audio file) when it is really another type of file (like a text file).

It's only when you change the file extension back to it's original file extension that you can open it and see the original file.
