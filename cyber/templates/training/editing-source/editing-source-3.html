Oops, we broke the mini-challenge button! Go ahead, try and click it- it won't work

The good news is you can fix it.

Let's learn how

In the previous training we learnt what a tag was, and how everything you see on a website is made up of lots of different tags. Paragraphs are made using the <p> tag, images use the <img> tag, buttons use <button>, forms use <form>, etc... If you want to learn more about HTML, w3schools is a great resource.

Attributes of a tag

To make a button in HTML you need an opening <button> tag, the text you want inside your button, and finally a closing </button> tag. Here's an example:

<button >I am a button that does nothing</button>

There are different attributes that a tag can have. These are written after the element name (in this case, 'button') in the opening tag. Here's another example:

<button class="btn" name="mybutton" disabled>I am a button with attributes</button>

In this button, the attributes we have set its class, name, and disabled it. Most HTML tags have attributes, and these are very important for setting the properties of an element. For example, the width of an image is an attribute of the <img> tag

Give it a try

Try editing the source code of the buttons to change the class to class="btn-warning". You'll have to inspect the source code, double click on the <button> tag and type the new attributes you want. Watch the demo below to see how. You can also experiment with removing and adding the disabled attribute.

Mini-Challenge

Ruh roh! We broke the button! Can you edit the button's source code to enable it again?

The first step is to find the part of the source code that makes the "Accept Mini-Challenge" button. Try right clicking directly above the button and clicking "Inspect". You'll have to open up some div tags using the drop-down arrows to find the button. Then double click the button's tag to edit it; you will want to highlight the 'disabled' word and delete it
