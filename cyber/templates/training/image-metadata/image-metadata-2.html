So, you know about this mysterious hidden data we call EXIF metadata. Now how do we see it for ourselves

To have a look at image metadata we need to use tools/programs to examine the image for us. The easiest way for us to do this, is to use an online metadata viewer. There are many available, but we'll use this one: https://www.exifdata.com/

How to use it

Let's look at the metadata for this woman taking a selfie

Some common types of metadata:

  1. Right click the above image, and save it on your computer "Save as..."
  2. Navigate to https://www.exifdata.com/
  3. "Choose file" and find the photo you just saved
  4. Click Upload

If you've done this right, you should be able to see your first image metadata. We can see that the photo was created on 17th Jan 2019, as well as other technical information.

Try it yourself and see if you can find the 'author' of the photo.
