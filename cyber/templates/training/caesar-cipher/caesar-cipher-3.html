We now know how to make secret messages, great!

But that's no good to us if we can't decrypt them again. Luckily decrypting the caesar cipher is the same process as encrypting, but backwards

If we have the message "B" that has been encrypted with a caesar shift of 1, we move backwards one place in the alphabet to get "A"

Likewise if we receive the message "FDW" that has been encrypted with a caesar cipher of 3, we can decrypt it by taking each letter and moving it backwards 3 places in the alphabet

"F" becomes "C", "D" becomes "A", and "W" becomes "T". So our decrypted message is "CAT"!



We've received a mysterious secret message... Can you decrypt it? The caesar shift used to encrypt it is 5, so we must move each letter backwards 5 places.
