Congrats on making it through the training so far. We've got one more trick up our sleeve to show you.

What if you receive an encrypted caesar cipher message, but you don't know the shift? Even if you try out every possible shift using an online website, this will still take a long time.

Luckily, it's possible to 'brute force' the encrypted code. This means a website/tool will guess every possible shift for us. We're then able to see all of the results and see if any of the shifts we've tried resulted in a decrypted message that makes sense.

We can use the same website to do this: https://www.dcode.fr/caesar-cipher

Here's an example where we didn't know the caesar shift. You enter the encrypted message, then select the 'Brute Force' option, and click the Decrypt button.

Every possible caesar shift will be attempted, and it will detect the result with the most real words and display that version at the top. Here you can see the correct shift was 24 in this example

Try it out yourself now to decrypt the following with an unknown caesar shift:

Csy'zi ksx xlmw! Iewc tiewc

You'll want to use all your knowledge from this topic to decrypt a secret message for this challenge. Good luck!
