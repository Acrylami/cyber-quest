Security Pathway

This is a recommended lesson plan to teach password security, very basic penetration testing, and code breaking skills. It is best used as a combined theory and practical exercise when teaching security and cryptography, and will serve well as an introduction to these topics

Recommended Plan (50 mins-1.5 hrs):

  1. Caesar Cipher Training
  2. Substitution Cipher Training
  3. Manual Cracking Training

Recommended setup

Access to Google Chrome/Firefox/Microsoft Edge recommended. Preferably Chrome to ensure demonstations match the animations provided in the training.

More Info

The Caesar Cipher Training teaches the concepts of the Caesar Cipher from the basics, and gradually introduces more difficult concepts and the utilisation of external tools to brute force codes. This exercise would be suitable for both students with no prior knowledge, and those with experience

The Substitution Cipher training would likewise be suitable for students with experience or not, and introduces theory with regular practical challenges

The Manual Cracking training provides a basic introduction to password security, a very important topic to cover. It then hammers home the importance of choosing secure passwords with practical exercises in cracking a password form by 'brute forcing' popular passwords. This is an exciting activity simulating a real website.
