Web Pathway

This is a recommended lesson plan to teach web technologies including HTML and robots.txt. It is best used to serve as an introduction to the topics or to supplement existing knowledge to apply known theory

Recommended Plan (45 mins-1.5 hrs):

  1. Inspecting Source Code training
  2. Editing Source Code training
  3. Robots.txt training (optional)

Recommended setup

Access to Google Chrome/Firefox/Microsoft Edge recommended. Preferably Chrome to ensure demonstations match the animations provided in the training.

More Info

The Inspecting Source Code provides a basic introduction to HTML, the building block language of websites. By getting students to inspect the source code of this website and others, they will become familiar with the structure of HTML. This training exists as a practical beginners HTML activity, and will not teach the bredth of HTML theory. You may want to use this training at the end of a HTML lesson to test student knowledge

The Editing Source Code training takes the previous training a step further, by getting students to edit the source code of the website. If the previous "Inspecting Source Code" training has not been completed prior, some level of HTML experience is required. This exercise would also supplement existing HTML lessons well, or serve as an introduction to HTML code.

The Robots.txt training is a supercurricular activity, providing a background on webcrawlers and website layout. This activity would serve well as an extension activity or further reading type exercise
